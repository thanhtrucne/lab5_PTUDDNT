# Lab 5: Flutter Todo App with .NET Core Web API (JWT)

Dự án này là một ứng dụng quản lý công việc (Todo App) đầy đủ tính năng, bao gồm:
- **Frontend**: Ứng dụng di động được xây dựng bằng Flutter.
- **Backend**: RESTful API được xây dựng bằng ASP.NET Core Web API (.NET 9), sử dụng Entity Framework Core và SQLite.
- **Bảo mật**: Xác thực và phân quyền bằng JWT (JSON Web Tokens).
